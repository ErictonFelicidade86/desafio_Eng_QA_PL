# Automação de Teste com Cypress

## Sobre

Cypress Teste Automatizado de API

## Stacks
- Cypress
- Javascript
- Node.js

## Rodando

1. Clonar o repositório, instalar as dependências
```
npm install
```

2. Subir o Cypress UI
```
npx cypress open 
```

3. Executar testes em Headless
```
npx cypress run 
```
